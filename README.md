## Overview
This repository contains Dr. Prakash Rao Ragiri's academic profile webpage integrated with Google Gemini AI for dynamic responses.

### Features
- Responsive academic profile design
- Gemini AI chatbot with:
  - Real-time responses using Gemini API
  - Voice input (Web Speech API)
  - Speech synthesis
  - Fallback mode if API fails
- Citation graph visualization using Chart.js

## Setup Gemini API
1. Enable Gemini API in Google Cloud.
2. Get your API key.
3. Replace `YOUR_API_KEY` in `index.html`.

## Fallback Mode
If Gemini API fails, the chatbot provides predefined responses for research areas, publications, and actions.

## Deployment on GitHub Pages
1. Create a new public repository.
2. Upload `index.html` and `UploadedImage0.jpg`.
3. Enable GitHub Pages in Settings.
4. Access your site at:
```
https://<username>.github.io/<repository-name>/
```
